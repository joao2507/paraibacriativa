Facebook Optimize is a plugin to optimize your web pages to sharing on Facebook, adjusting the thumbnails and descriptions.


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

Go to Settings > Facebook Optimize and insert the requested info.

== Description ==

Facebook Optimize add to your site head the open graph tags necessary to improve the sharing on Facebook.
